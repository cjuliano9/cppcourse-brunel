double  input_current 
)

update the potential of neuron at each time step and return a boolean if spiking

Verify if neuron is refractory or not. If not: update the membrane potential with the background noise (poisson distribution) and the synaptic current from other neurons. Return true if the neuron is spiking, false if in refractory or inferior to threshold.
